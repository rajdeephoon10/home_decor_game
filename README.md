# AI Interior Decor – Unity Project

# Overview

This project is a simple interactive 3D room where users can experiment with interior design.
Users can change wall colors and get AI-generated design suggestions based on their style preference.

# Features

* Interactive 3D room environment
* Wall color customization using UI buttons
* User input field for style preferences
* AI-generated interior design suggestions

# AI Integration

The user enters a style such as **luxury** or **modern**.
The application sends this input to an AI model using an API.
The AI returns multiple interior design suggestions.

# Technologies Used

* Unity Engine
* C#
* TextMeshPro UI
* AI API integration

# How to Run

1. Download the Build folder or `.exe`.
2. Run the application.
3. Enter a style (example: `luxury`).
4. Click **Suggest Design**.
5. AI suggestions will appear in the UI.
